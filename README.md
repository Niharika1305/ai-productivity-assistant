# AI Productivity Assistant

## Project Description

This project is an AI Productivity Assistant built using Python and Groq API. It accepts user tasks, generates structured productivity guidance using AI, and saves responses into a text file.

## Features

* User input support
* Groq API integration
* Environment variable security using .env
* Structured AI responses
* Automatic file saving
* Output history storage

## Tech Stack

* Python
* Groq API
* python-dotenv
* VS Code

## Setup Instructions

1. Clone the repository
2. Create virtual environment
3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Create .env file

```env
GROQ_API_KEY=your_api_key
MODEL_NAME=llama-3.3-70b-versatile
```

## Environment Variables

* GROQ_API_KEY
* MODEL_NAME

## How to Run

```bash
python main.py
```

Create a 3-day study plan for Python functions
